Projects

Pricatch

Pricatch is a web application that keeps track of prices of products that you wish to buy. It will send you an email notification when the price of product you asked to track gets below a certain price that you entered.

This project is broken up into a backend and frontend. The backend contains the Django project which uses the Django Rest Framework to host a simple API. The frontend uses React and queries data from the API using the Axios module.

Writeasy

Writeasy is a one stop mobile application for all the aspiring writers. It features various elements that allows for the writer to organise their thoughts and various elements of their writing projects. You dont have to worry ever about finding your pointers cause you have them all in one app.

This project uses React Native for the frontend of application. It stores the data in a local SQLite database making it a completely offline app.
